# Projet Notes Étudiants

Étude de la performance des étudiants (Kaggle dataset).

## Structure
- `data/`: Données brutes et nettoyées.
- `notebooks/`: Étapes d'analyse et de clustering.
- `src/`: Code source modulaire.
- `outputs/`: Résultats et graphiques.
- `rapport/`: Rapport final.

## Installation
```bash
pip install -r requirements.txt
```
